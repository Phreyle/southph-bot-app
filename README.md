
# SouthPH Bot

SouthPH Bot is a private, modular Discord bot built for personal use. It features interactive games like rock-paper-scissors, a simple virtual banking system, and a flexible command structure. The bot is designed for fun and experimentation within a small Discord community.

## Features
- Virtual bank (balance, deposit, withdraw)
- Slash commands and interactive components
- Easily extendable with new commands and features

This bot is not intended for public distribution or large-scale deployment.
